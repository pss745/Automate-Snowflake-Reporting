# Automate-Snowflake-Reporting
Query your Snowflake data and automatically send the report to email

Python Script.py 
Set your Snowflake credentials and give your SQL code in this.
The SQL code is converted to a pandas Dataframe which is then converted to a csv

Script.py
This has the code to send the above csv to any email.
Specify the email from,to, subject etc.

The code then gets the report and attaches this to email and sends it.
